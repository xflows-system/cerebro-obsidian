## O que é o Odysseus

Projeto open source lançado pelo PewDiePie no GitHub. É um **assistente pessoal de IA auto-hospedado** — uma alternativa ao ChatGPT/Claude.ai que roda na sua VPS ou home lab. Você controla os dados, escolhe o modelo, sem depender de nenhuma big tech.

## Features demonstradas

| Feature | O que faz |
|---|---|
| **Multi-model** | Conecta DeepSeek, OpenRouter, modelos locais, Anthropic direto |
| **Deep Research** | Pesquisa autônoma com múltiplas fontes, output em relatório |
| **Model Comparison** | Compara modelos lado a lado (blind, parallel, shuffle) |
| **Brain** | Memória persistente — extrai memórias e skills automaticamente das conversas |
| **Tasks** | Agentes agendados: Email Summary, Calendar Classify, etc. |
| **Library** | Documentos e pesquisas salvas por sessão |
| **Email** | Integração com email (IMAP — mostrou erro na demo) |

## Proposta de valor central

1. **Independência de big tech** — se a Anthropic cortar seu acesso, você continua com o modelo que quiser
2. **Dados ficam na sua máquina** — compliance, advogados, dados sigilosos
3. **Um lugar só** para todos os modelos — sem múltiplas assinaturas

## Avaliação do criador (mano deyvin)

- **Ponto mais forte:** memória persistente que aprende quem você é sem importar nada

## Vale a pena para quem já tem stack própria?

**Não.** O Odysseus resolve para quem ainda não tem infraestrutura de IA própria. Quem já tem VPS + orquestrador + multi-model já vai além do que o Odysseus oferece. As features dele (memória, pesquisa, tasks agendados, multi-model) já existem em stacks mais avançadas com mais controle.
