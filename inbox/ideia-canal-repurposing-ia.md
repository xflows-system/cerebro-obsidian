# Ideia: Canal YouTube — Repurposing com IA

**Data:** 2026-06-19
**Origem:** reel @oandersonribeiro01 (Instagram)
**Status:** para testar

---

## O Método (do criador)

1. Escolher nicho com vídeos virais: finanças, emagrecimento, relacionamento
2. Pegar vídeo com +1M views no YouTube
3. Transcrever com IA
4. Gerar áudio com outra IA (TTS/voz sintética)
5. Montar vídeo com outra IA (juntar áudio + imagens/vídeo)
6. Subir no YouTube com SEO + thumbnail
7. Monetizar: AdSense por views + produto no final

---

## Stack disponível (sem custo extra)

| Passo | Ferramenta |
|---|---|
| Transcrição | Groq Whisper (chave ativa) |
| TTS | ElevenLabs free ou HuggingFace |
| Montagem | ffmpeg + imagens (gratuito) |
| SEO | Hermes com skill |
| Upload | API YouTube |

---

## MVP para testar

- Escolher 1 nicho + 1 vídeo viral específico
- Transcrever com Groq Whisper  
- Reescrever com IA (não copiar direto — risco copyright)
- Gerar áudio TTS com voz diferente
- Montar com ffmpeg (slides + áudio)
- Subir e observar 7 dias de resultado

**Tempo estimado primeiro vídeo:** 2–3h

---

## Riscos

- YouTube derruba conteúdo duplicado — precisa reescrever, não só re-narrar
- Monetização só libera com 1.000 inscritos + 4.000h
- O criador do reel não mostrou prova real de resultado
